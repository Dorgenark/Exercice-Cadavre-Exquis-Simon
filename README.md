# Le dernier des dresseurs d'escargot

Intéressons-nous à l’histoire poignante de cet homme Vietnamo-Moldave pour qui rien de réussissait, et qui décida, dans un éclair de génie ou un hasard du destin, à se lancer sur la trace de ses ancêtres pour devenir dresseur d’escargot. Tout commença en décembre 1976.
